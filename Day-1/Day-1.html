  9. What is Github and how to use it?

    10. = This is used to create a online backup. For that we have to login at github and have to create a new repository. After that there are some steps for adding files in github. Steps are given bellow: (from the VS Coad)

    1. Ctrl+J (to open the command promt)
    2. git init (to initialize the git)
    3. git add . (to add all files)
    4. git commit -m "a meaningfull comment" (to commit added files)
    5. git branch -M main (to set the branch to main)
    6. git remote add origin + the link (to add the origin)
    7. git push -u origin main

    After completing those steps our file will be visible to our new repository that we created on github

    N.B: If we do this for first time we have to face the authentication from github. The process will be displayed on the command prompt. After first time we may use three commands to make update to our project. Commands are:
      i. git add .
     ii. git commit -m "a meaningfull comment"
    iii. git push
